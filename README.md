Grady
=
Personal framework for quick styling of site.
-
Components:
 - Input fields
 - Buttons
 - Checkboxes
 - Radio buttons
 - Error message areas
 - Responsive menus

```
	|   └── grady.min.css
	|   
	|
	├── demo/
	|   ├── demo.css
	|   └── grady_docs.html
	|
	└── js/
	    ├── grady.js
	    └── grady.min.js
```
